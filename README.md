# 結マール5 公式ホームページ

一般社団法人ベースボール沖縄が運営する Baseball5 チーム「結マール5」の公式ホームページです。

## 公開方法

このフォルダを GitHub リポジトリのルートに配置すると、GitHub Pages で公開できます。

1. GitHub で新しいリポジトリを作成
2. このフォルダ内のファイルをすべてアップロード
3. GitHub の `Settings` → `Pages` を開く
4. `Deploy from a branch` を選択
5. Branch を `main`、Folder を `/root` に設定

## 入口ページ

- `index.html` がトップページです。
- 大会ページ、子ども向けイベント、学校教育、スポンサー資料ページも同じフォルダ内に含まれています。

## 注意

- 画像や動画は `assets/` フォルダ内にあります。
- ファイル名やフォルダ構成を変更すると、ページ内の画像リンクが切れる場合があります。
